# Addiction & Recovery: A Jungian Framework

## Jung's Core Understanding

### The Spiritual Thirst

Jung viewed addiction radically differently from his contemporaries. Rather than mere mood alteration, substances work at a deep level of psychic functioning:

```
WHAT SUBSTANCES DO (Jungian View)
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   FRAGMENTED PSYCHE              INTOXICATED STATE          │
│   ┌─────┐ ┌─────┐               ┌───────────────────┐      │
│   │ Ego │ │Shadow│              │                   │      │
│   └──┬──┘ └──┬──┘    ────►     │   ILLUSORY        │      │
│      │       │                  │   WHOLENESS       │      │
│   ┌──┴──┐ ┌──┴──┐              │                   │      │
│   │Parts│ │Pain │              │   (numinous,      │      │
│   └─────┘ └─────┘              │    compelling)    │      │
│                                 └───────────────────┘      │
│                                                             │
│   Splits, fragments,            Boundaries blur,           │
│   disconnection                 fragments merge            │
│                                                             │
└─────────────────────────────────────────────────────────────┘

THE PROBLEM: This wholeness dissolves upon sobering.
             The search to repeat it becomes compulsive.
```

### Spiritus Contra Spiritum

Jung's formula from his 1961 letter to Bill Wilson:

> "Alcohol in Latin is 'spiritus' and you use the same word for the highest
> religious experience as well as for the most depraving poison. The helpful
> formula therefore is: spiritus contra spiritum."

```
SPIRITUS CONTRA SPIRITUM
═══════════════════════════════════════════════════════════════

     LOW SPIRIT                      HIGH SPIRIT
     (Substance)                     (Genuine Experience)
         │                                  │
         ▼                                  ▼
    ┌─────────┐                      ┌─────────────┐
    │ Alcohol │                      │ Spiritual   │
    │ Drugs   │   ◄── COUNTERED BY ──►│ Awakening   │
    │ Behavior│                      │ Wholeness   │
    └─────────┘                      └─────────────┘
         │                                  │
         ▼                                  ▼
    Illusory wholeness              Genuine integration
    Temporary relief                Lasting transformation
    Increasing fragmentation        Progressive individuation

THE CRAVING IS LEGITIMATE — only the method is destructive.
```

## Practical Techniques

### Active Imagination with Craving

```
DIALOGUING WITH THE CRAVING
═══════════════════════════════════════════════════════════════

STEP 1: INVOCATION
        ┌─────────────────────────────────────┐
        │ Feel the craving in your body.      │
        │ Where does it live? What does it    │
        │ feel like? Allow it to take form.   │
        └─────────────────────────────────────┘
                        │
                        ▼
STEP 2: PERSONIFICATION
        ┌─────────────────────────────────────┐
        │ Let the craving become a figure.    │
        │ What does it look like? How old?    │
        │ What is its posture, expression?    │
        └─────────────────────────────────────┘
                        │
                        ▼
STEP 3: DIALOGUE
        ┌─────────────────────────────────────┐
        │ Ask: "What do you really want?"     │
        │      "What are you protecting?"     │
        │      "What do you need from me?"    │
        │ Listen without judgment.            │
        └─────────────────────────────────────┘
                        │
                        ▼
STEP 4: DISCOVERY
        ┌─────────────────────────────────────┐
        │ Beneath the craving is a           │
        │ legitimate need. Name it.           │
        │ How might this need be met         │
        │ without the substance?              │
        └─────────────────────────────────────┘
                        │
                        ▼
STEP 5: INTEGRATION
        ┌─────────────────────────────────────┐
        │ Thank the figure. Acknowledge its   │
        │ purpose. Commit to addressing the   │
        │ underlying need directly.           │
        └─────────────────────────────────────┘
```

## Integration with 12-Step Programs

```
JUNGIAN CONCEPTS IN THE 12 STEPS
═══════════════════════════════════════════════════════════════

STEP 1: Powerlessness          = Ego deflation
STEP 2: Higher Power           = Recognition of Self
STEP 3: Decision               = Beginning ego-Self surrender
STEP 4: Moral Inventory        = Shadow work begins
STEP 5: Admission              = Shadow made conscious
STEP 6: Ready for removal      = Willingness to integrate
STEP 7: Humbly asked           = Ego in right relationship
STEP 8: List of harms          = Projection withdrawn
STEP 9: Amends                 = Shadow integration in action
STEP 10: Continued inventory   = Ongoing shadow awareness
STEP 11: Meditation/prayer     = Cultivating ego-Self axis
STEP 12: Spiritual awakening   = Individuation milestone

Jung's influence on AA is direct: his patient Rowland H.
found recovery through spiritual experience, then helped
found AA. The 12 Steps are essentially Jungian individuation
in accessible form.
```

## Cautions and Boundaries

```
WHEN JUNGIAN WORK IS CONTRAINDICATED
═══════════════════════════════════════════════════════════════

✗ Active intoxication or withdrawal
  └── Stabilization first; depth work requires stability

✗ Acute psychosis or severe dissociation
  └── Risk of ego dissolution; need containment

✗ Without adequate support system
  └── Shadow work can destabilize; need human anchors

✗ As replacement for evidence-based treatment
  └── Complement, don't replace medical/behavioral care

✗ Without willingness
  └── Forced depth work is traumatizing, not healing

WHEN TO REFER TO HUMAN ANALYST:
├── Overwhelming affect from exercises
├── Persistent intrusive symptoms
├── History of trauma requiring containment
├── Desire for relational depth work
└── When something feels "too big"

RESOURCES:
├── IAAP (International Association for Analytical Psychology)
├── C.G. Jung Institute (various cities)
├── SAMHSA National Helpline: 1-800-662-4357
└── Local addiction medicine specialists
```
