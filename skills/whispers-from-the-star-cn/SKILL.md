# 星之低语 🌌

一个沉浸式的科幻生存冒险游戏，玩家将扮演坠落在陌生星球的宇航员 Stella Chen，在盖亚星球上展开一场惊心动魄的生存与探索之旅。

## 游戏简介

**背景设定：**
- 🚀 主角：Stella Chen，中国宇航员，30岁
- 🌍 地点：盖亚（Gaia）星球 - 一个美丽而危险的外星世界
- ⚠️ 困境：飞船坠毁，通讯中断，孤身一人
- 🎯 目标：在陌生星球上生存，寻找回家之路

**游戏特色：**
- 丰富的科幻设定和世界观
- 真实的生存压力（食物、水源、庇护所）
- 多样的探索场景（洞穴、废墟、森林）
- 神秘的外星生命互动
- 科学与冒险的完美结合

## 核心场景

游戏包含 13 个主要场景，每个都提供独特的挑战和体验：

### 生存基础
- **shelter-finding** 🏠 - 寻找庇护所：第一夜的紧迫挑战
- **water-crisis** 💧 - 水源危机：寻找安全的饮用水
- **cooking-adventure** 🍳 - 烹饪冒险：辨识外星食材，创造美食

### 探索发现
- **cave-exploration** 🕳️ - 洞穴探险：深入未知的地下世界
- **ancient-ruins** 🏛️ - 古代遗迹：探索外星文明的痕迹
- **alien-flora** 🌿 - 异星植物：研究奇特的外星生态

### 危机事件
- **storm-survival** ⛈️ - 风暴生存：面对致命的外星天气
- **power-crisis** ⚡ - 能源危机：设备失灵，寻找替代能源

### 重大突破
- **alien-encounter** 👽 - 外星接触：人类历史性的第一次接触
- **distress-signal** 📡 - 求救信号：尝试与外界建立联系
- **signal-tower** 📶 - 信号塔：建造通讯设施
- **escape-plan** 🚁 - 逃离计划：策划回家之路

### 日常探索
- **daily-science** 🔬 - 日常科研：记录观察，科学探索

## 使用方法

### 开始新游戏

推荐从基础生存场景开始：

```
read references/shelter-finding.md  # 第一晚：寻找庇护所
```

### 场景推进

根据玩家的选择和探索方向，读取对应场景：

```
read references/water-crisis.md      # 解决饮水问题
read references/cave-exploration.md  # 探索洞穴
read references/alien-encounter.md   # 遭遇外星生命
```

### 游戏流程建议

**第一阶段：生存建立（Day 1-3）**
1. shelter-finding - 找到安全的栖身之所
2. water-crisis - 解决饮水问题
3. cooking-adventure - 解决食物来源

**第二阶段：探索扩展（Day 4-7）**
4. cave-exploration - 探索洞穴资源
5. alien-flora - 研究植物生态
6. daily-science - 科学记录观察

**第三阶段：危机应对（Day 8-10）**
7. storm-survival - 经历风暴考验
8. power-crisis - 解决能源问题

**第四阶段：重大发现（Day 11+）**
9. ancient-ruins - 发现外星文明
10. alien-encounter - 与智慧生命接触
11. distress-signal - 发送求救信号
12. signal-tower - 建立通讯
13. escape-plan - 策划回家

### 3. 环境互动
- 🌤️ 动态天气系统
- 🌓 昼夜循环
- 🐾 野生动物遭遇
- 🌿 植物采集和种植

### 4. 技能成长
- 🔧 工程技能（修理、建造）
- 🥾 生存技能（狩猎、采集）
- 🧠 科学技能（研究、分析）

## 角色设定

**Stella Chen - 主角**
- 年龄：30岁
- 职业：中国航天局宇航员
- 专长：天体物理学、工程学
- 性格：理性冷静、坚韧乐观、好奇心强
- 背景：经验丰富的宇航员，第一次深空任务

**心理状态变化：**
- Day 1-3：震惊、恐惧、求生本能
- Day 4-7：适应、好奇、科学兴趣
- Day 8+：希望、决心、探索欲望

## 盖亚星球设定

**环境特征：**
- 🌍 类地行星，重力约为地球 0.9 倍
- 🌡️ 温带气候，昼夜温差大
- 💨 大气成分类似地球但氧含量略高
- 🌊 丰富的水资源（河流、湖泊）
- 🌲 茂密的森林和多样化生态系统

**生物特征：**
- 发光植物（夜间照明）
- 晶体结构的植物
- 群居的小型动物
- 神秘的大型生物
- 可能存在的智慧生命

**危险因素：**
- 强烈的电磁风暴
- 有毒的植物和动物
- 地质活动（地震、火山）
- 未知的疾病和病原体

## 叙事风格

### 第一人称日志体
以 Stella 的视角进行叙事，营造沉浸感：
- 使用第一人称："我发现..."
- 包含心理活动和情感

- 每个场景 300-500 字
- 等待玩家做出选择后继续
- 环境细节丰富但不冗长
- 保持紧张感和好奇心

3. **情感共鸣** - Stella 的孤独、恐惧、希望
4. **探索自由** - 允许玩家自主选择方向
5. **意外惊喜** - 设计突发事件和发现
