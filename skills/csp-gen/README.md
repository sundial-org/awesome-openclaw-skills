# CSP Generator

Generate Content Security Policy headers for your site.

## Quick Start

```bash
npx ai-csp https://mysite.com
```

## What It Does

- Analyzes your site and generates CSP headers
- Handles all directives automatically
- Both strict and relaxed policy options
