# Landing Page Generator

Generate a beautiful HTML landing page from your package.json in one command.

## Quick Start

```bash
npx ai-landing .
```

## What It Does

- Reads package.json and generates a complete landing page
- Single HTML file output, deploy anywhere
- Responsive design built in
- Proper meta tags and semantic HTML

## Options

```bash
npx ai-landing [dir] -o <output-file>
```
