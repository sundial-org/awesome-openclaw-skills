# Fizzy CLI

This repo contains a standalone CLI.

## Install

```bash
npm i -g @emredoganer/fizzy-cli
```

## Auth

Generate a Personal Access Token in Fizzy:

```bash
fizzy auth login
```

(You will be prompted for your token.)
