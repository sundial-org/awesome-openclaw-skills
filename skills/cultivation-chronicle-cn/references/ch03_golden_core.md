# 金丹期 - 金丹凝成

## 境界概述

**金丹期（寿元：500-800年）**

- **修炼方式**：凝炼金丹，吞吐日月精华，神识外放
- **境界特征**：金丹成则寿元大增，可以神识感知方圆百里
- **突破试炼**：破丹成婴，承受天劫洗礼
- **人生体验**：见证筑基期道友陨落，感悟生死无常，权衡恩怨情仇

**金丹期分层**：初期 → 中期 → 后期 → 圆满

## 主要任务

1. **境界修炼**：金丹期各层次提升
2. **神识修炼**：神识外放，感知天地
3. **秘境探索**：上古遗迹、大能传承
4. **法宝争夺**：灵阶、玄阶法宝竞争
5. **人性主题**：生死无常，道友陨落，恩怨情仇

## 金丹期奇遇

- 上古剑冢
- 大能洞府
- 宗门战争
- 击杀妖兽（金丹级）
- 法宝炼制

## 突破元婴试炼

**破丹成婴劫**：
- 金丹化婴，元婴出窍
- 承受天劫
- 成功：进入元婴期（ch04_nascent_soul）
