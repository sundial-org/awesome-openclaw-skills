# Barretenberg Reference

## Scope

Use this reference when using Barretenberg as the proving backend with Noir or when the user asks about `bbup`, `bb`, or `bb.js`.

## Documentation

Read https://barretenberg.aztec.network/docs/ for usage and API details.

## Version Compatibility

Before pinning any versions of `bb` or `bb.js`, run `bbup` to install the Barretenberg version compatible with the local Noir toolchain and use that to determine versioning.
