# Auth Auditor

Audit your auth implementation for security flaws.

## Quick Start

```bash
npx ai-auth-check src/
```

## What It Does

- Checks password hashing, CSRF protection, session security
- Detects JWT misuse and insecure configurations
- Reports issues with fix instructions
