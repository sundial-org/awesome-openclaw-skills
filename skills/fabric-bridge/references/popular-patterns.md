# Popular Fabric Patterns

Curated high-quality patterns by category. Run with `fabric-ai -p <name> -s`.

## Content Analysis & Extraction

- `extract_wisdom` — Flagship pattern. Extract IDEAS, INSIGHTS, QUOTES, HABITS, FACTS from any content.
- `analyze_claims` — Fact-check framework. Rate claims by evidence strength.
- `find_logical_fallacies` — Detect logical fallacies with detailed explanations (222 rules).
- `summarize` — Concise content summary.
- `extract_main_idea` — Pull the core argument from text.
- `extract_article_wisdom` — Article-focused wisdom extraction.
- `label_and_rate` — Rate content quality and assign labels.
