# Lesson 2 Vocabulary (From PDF)

## Objects (Mono)
- **Hon (本)**: Buku / Book
- **Kagi (鍵)**: Kunci / Key
- **Enpitsu (鉛筆)**: Pensil / Pencil
- **Jisho (辞書)**: Kamus / Dictionary
- **Shāpu penshiru (シャープペンシル)**: Pensil mekanik / Mechanical Pencil
- **Zasshi (雑誌)**: Majalah / Magazine
- **Meishi (名刺)**: Kartu nama / Business Card
- **Kasa (傘)**: Payung / Umbrella
- **Kaban (かばん)**: Tas / Bag
- **Nōto (ノート)**: Buku catatan / Notebook
- **Keshigomu (消しゴム)**: Penghapus / Eraser
- **Bōrupen (ボールペン)**: Pulpen / Ballpoint Pen
- **Tokei (時計)**: Jam / Watch, Clock
- **Kuruma (車)**: Mobil / Car
- **Tsukue (机)**: Meja / Desk
- **Isu (椅子)**: Kursi / Chair
- **Omiyage (お土産)**: Oleh-oleh / Souvenir

## Kanji List
- **Hito (人)**: Orang / Person
- **Te (手)**: Tangan / Hand
- **Yama (山)**: Gunung / Mountain
- **Kawa (川)**: Sungai / River
- **Onna (女)**: Perempuan / Woman
- **Otoko (男)**: Laki-laki / Man
- **Ue (上)**: Atas / Up, Above
- **Shita (下)**: Bawah / Down, Below
- **Hairu (入る)**: Masuk / To enter

## Phrases
- **Korekara osewa ni narimasu**: Mohon bimbingannya mulai sekarang / Thank you for your support from now on.
- **Douzo yoroshiku onegaishimasu**: Senang bertemu denganmu (Formal).
