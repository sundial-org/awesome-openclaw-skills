# Coverage Booster

Find untested code paths and generate tests to boost coverage.

## Quick Start

```bash
npx ai-coverage-boost src/
```

## What It Does

- Finds untested functions, branches, and lines
- Generates Jest/Vitest compatible test files
- Prioritizes by complexity
