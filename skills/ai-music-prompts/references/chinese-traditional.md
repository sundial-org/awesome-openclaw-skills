## 古风音乐 / Gufeng Music

### 什么是古风 / What is Gufeng

**古风**（Gufeng）是一种融合中国传统元素的现代音乐风格，常见于：
- 仙侠/武侠影视配乐
- 网络游戏背景音乐
- 古风歌曲翻唱
- 汉服文化活动

**特点**：
- 使用中国传统乐器（古筝、笛子、二胡等）
- 五声音阶（宫商角徵羽）
- 古诗词歌词或仿古文风
- 意境优美、诗意浓厚

### 古风提示词模板 / Gufeng Prompt Templates

#### 基础模板 / Basic Template
```
古风 [情绪] [速度] [乐器] [人声] - [意境描述]

Gufeng [mood] [tempo] [instruments] [vocals] - [atmosphere description]
```

#### 仙侠风格 / Xianxia Style
```
古风 空灵 70-80 BPM 古筝笛子 悠扬女声 - 仙侠风格，清幽脱俗的意境，如仙境般美丽

Gufeng ethereal 70-80 BPM guzheng dizi melodious female vocals - Xianxia (fantasy) style with pure and transcendent atmosphere, beautiful as fairyland
```

#### 武侠风格 / Wuxia Style
```
古风 激昂 120-140 BPM 中国鼓琵琶 高亢男声 - 武侠风格，豪迈壮烈，江湖恩怨

Gufeng passionate 120-140 BPM Chinese drum pipa powerful male vocals - Wuxia (martial arts) style, heroic and intense, Jianghu conflicts
```

#### 宫廷风格 / Palace Style
```
古风 华丽 90-100 BPM 古筝扬琴 宏大合唱 - 宫廷风格，华丽典雅，皇家气派

Gufeng magnificent 90-100 BPM guzheng yangqin grand choir - Palace style, magnificent and elegant, royal grandeur
```

## 国风音乐 / Guofeng Music

### 什么是国风 / What is Guofeng

**国风**（Guofeng）是现代元素与中国传统元素的融合风格，常见于：
- 流行歌曲的中国风变奏
- 影视配乐
- 广告音乐
- 文化活动

**特点**：
- 传统乐器 + 现代编曲
- 五声音阶 + 现代和声
- 中文歌词 + 现代主题
- 东方韵味 + 西方制作

### 国风提示词模板 / Guofeng Prompt Templates

#### 流行国风 / Pop Guofeng
```
国风 欢快 110-120 BPM 古筝电子乐 清新女声 - 现代国风，传统与现代融合

Guofeng upbeat 110-120 BPM guzheng electronic fresh female vocals - Modern Guofeng, fusion of tradition and modern
```

#### 影视国风 / Cinematic Guofeng
```
国风 宏大 100-110 BPM 中国鼓弦乐 气势磅礴 - 影视配乐风格，大气磅礴，史诗感

Guofeng grand 100-110 BPM Chinese drum strings majestic - Cinematic style, magnificent and epic
```

#### 电子国风 / Electronic Guofeng
```
国风 前卫 130-140 BPM 古筝合成器 混合人声 - 电子国风，未来感与传统结合

Guofeng avant-garde 130-140 BPM guzheng synthesizer mixed vocals - Electronic Guofeng, future meets tradition
```

## 中国传统乐器 / Chinese Traditional Instruments

### 弦乐器 / String Instruments

#### 古筝 / Guzheng (Chinese Zither)
```
特点：悠扬、清雅、流水般
情绪：空灵、悠扬、古典、宁静
提示词示例：
- 悠扬古筝，如流水潺潺
- 清雅古筝，古朴典雅
- 激昂古筝，气势磅礴
```

#### 二胡 / Erhu (Chinese Fiddle)
```
特点：哀婉、深情、诉说感
情绪：伤感、怀旧、深情、沧桑
提示词示例：
- 哀婉二胡，如泣如诉
- 深情二胡，缠绵悱恻
- 欢快二胡，喜庆热闹
```

#### 琵琶 / Pipa (Chinese Lute)
```
特点：清脆、力度丰富、颗粒感
情绪：刚柔并济、古典、优雅
提示词示例：
- 清脆琵琶，珠落玉盘
- 激昂琵琶，十面埋伏
- 柔美琵琶，春江花月
```

#### 扬琴 / Yangqin (Chinese Dulcimer)
```
特点：清脆、共鸣好、和声
情绪：明亮、欢快、和谐
提示词示例：
- 清脆扬琴，叮咚悦耳
- 欢快扬琴，喜庆祥和
```

### 管乐器 / Wind Instruments

#### 笛子 / Dizi (Chinese Bamboo Flute)
```
特点：清脆、悠远、穿透力
情绪：清幽、飘逸、自然、悠扬
提示词示例：
- 悠扬笛子，山间清风
- 清脆笛子，鸟鸣山幽
- 激昂笛子，万马奔腾
```

#### 箫 / Xiao (Vertical Bamboo Flute)
```
特点：低沉、幽深、空灵
情绪：沧桑、悠远、空灵、宁静
提示词示例：
- 低沉箫声，幽谷回响
- 空灵箫声，月夜静思
```

#### 唢呐 / Suona (Chinese Oboe)
```
特点：高亢、嘹亮、喜庆
情绪：热烈、喜庆、豪迈、悲壮
提示词示例：
- 高亢唢呐，锣鼓喧天
- 悲壮唢呐，如泣如诉
```

### 打击乐器 / Percussion Instruments

#### 中国鼓 / Chinese Drum
```
特点：震撼、有力、节奏感
情绪：宏大、磅礴、激昂、震撼
提示词示例：
- 震撼中国鼓，雷霆万钧
```

## 五声音阶 / Pentatonic Scale

### 中国五声 / Chinese Pentatonic

```
五声音阶：宫、商、角、徵、羽
对应音级：1、2、3、5、6（Do、Re、Mi、Sol、La）

特点：
- 没有 4（Fa）和 7（Ti）
- 音程和谐，旋律流畅
- 东方韵味浓郁
```

### 使用五声音阶的提示词 / Pentatonic Prompt Examples

```
古风 五声音阶 悠扬 古筝笛子 - 传统的五声音阶旋律

Gufeng pentatonic scale melodious guzheng dizi - Traditional pentatonic melody
```

## Mureka V8 古风优化技巧 / Mureka V8 Gufeng Optimization Tips

### ✅ 推荐做法 / Recommended

1. **明确风格**：古风、国风、仙侠、武侠
2. **指定乐器**：古筝、笛子、二胡等中国传统乐器
3. **描述意境**：清幽、空灵、悠扬、激昂
4. **提及主题**：修仙、江湖、宫廷、山水
5. **速度适中**：70-120 BPM 最为常见

### ❌ 避免错误 / Avoid

1. ❌ 乐器冲突：古风电吉他重金属（除非明确融合）
2. ❌ 速度过快：古风 180 BPM（除非特殊需求）
3. ❌ 情绪不符：古风欢快蹦迪（除非创意需求）
4. ❌ 元素过多：超过 1024 字符限制
5. ❌ 描述模糊：古风好听（太简单）
