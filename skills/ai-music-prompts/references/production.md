## Production Problem Solving / 制作问题解决

### Common Issues & Solutions / 常见问题与解决方案

#### Issue: Muddy Mix / 问题：混音浑浊
```
Cause / 原因：
Too much low-mid frequency overlap
太多中低频重叠

Solution / 解决方案：
"Clean up muddy frequencies, tighten low end"
CN: "清理浑浊频率，收紧低频"

Prompt Fix / 提示词修复：
"Add: 'clear frequency balance with tight, punchy low end'"
CN: "添加：'清晰的频率平衡，紧凑有力的低频'"
```

#### Issue: Thin Sound / 问题：声音单薄
```
Cause / 原因：
Lack of frequency range
缺乏频率范围

Solution / 解决方案：
"Add full frequency range with rich mids and deep bass"
CN: "添加完整频率范围，配丰富中频和深沉低频"

Prompt Fix / 提示词修复：
"Add: 'rich full-range sound with warm mids and deep bass'"
CN: "添加：'丰富全频声音，温暖中频和深沉低频'"
```

#### Issue: Vocals Buried / 问题：人声埋没
```
Cause / 原因：
Instruments too loud or vocals too quiet
乐器太大声或人声太小声

Solution / 解决方案：
"Bring vocals forward, balance mix"
CN: "把人声前移，平衡混音"

Prompt Fix / 提示词修复：
"Add: 'lead vocals upfront and clear in mix'"
CN: "添加：'主唱人声在前且在混音中清晰'"
```

#### Issue: Harsh Highs / 问题：刺耳高频
```
Cause / 原因：
Excessive high frequency content
过多高频内容

Solution / 解决方案：
"Soften high frequencies, add warmth"
CN: "软化高频，添加温暖"

Prompt Fix / 提示词修复：
"Add: 'warm, smooth high frequencies without harshness'"
CN: "添加：'温暖平滑高频，无刺耳感'"
```
