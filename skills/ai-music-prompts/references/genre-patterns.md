# Genre-Specific Prompt Patterns / 流派特定提示词模式

Detailed prompt structures and patterns for each major music genre, with Chinese translations.
每个主要音乐流派的详细提示词结构和模式，配有中文翻译。

## Pop / 流行音乐

### Standard Pop Structure / 标准流行结构
```
[Subgenre] [Tempo] [Key Elements] [Vocal Style] - [Theme/Description]

Template / 模板：
Pop [upbeat/ballad/mid-tempo] [BPM] [instruments] [vocal type] - [song theme]

Examples / 示例：

1. Upbeat Pop / 欢快流行：
"Pop upbeat 120-128 BPM synth bass drums catchy hooks bright female vocals - Feel-good anthem about self-empowerment with singalong chorus"

CN: "流行 欢快 120-128 BPM 合成器贝斯 鼓 朗朗上口的钩子 明亮女声 - 关于自我赋权的积极颂歌，有合唱部分"

2. Pop Ballad / 流行抒情：
"Pop ballad slow 70 BPM piano strings emotional vocals - Heartbreaking love song with powerful chorus"

CN: "流行 抒情 缓慢 70 BPM 钢琴 弦乐 情感人声 - 心碎情歌，有强力副歌"

3. Dance Pop / 舞曲流行：
"Pop EDM 128 BPM driving bass synths energetic vocals - High-energy dance track with euphoric drops"

CN: "流行 EDM 128 BPM 驱动贝斯 合成器 充满活力人声 - 高能量舞曲，有令人振奋的节奏"

### K-Pop Pattern / K-Pop 模式
```
K-Pop [upbeat/mid-tempo] [BPM] [elements] [idol style] - [concept]

Example / 示例：
"K-Pop upbeat 130 BPM electronic elements catchy hooks idol vocals high energy - Modern K-pop track with dynamic production and bilingual hooks"

CN: "K-Pop 欢快 130 BPM 电子元素 朗朗上口的钩子 偶像人声 高能量 - 现代 K-Pop 轨道，有动态制作和双语钩子"
```

### Chinese Pop Pattern / 华语流行模式
```
Mandarin Pop [tempo] [instruments] [mood] [vocal style] - [theme]

Example / 示例：
"华语流行 95 BPM 钢琴 弦乐 深情 女声 - 现代华语流行情歌，有优美旋律和真挚歌词"

Translation:
"Mandarin Pop 95 BPM piano strings sentimental female vocals - Modern Chinese pop love song with beautiful melodies and heartfelt lyrics"
```

## Mood Mapping by Genre / 各流派情绪映射

### Pop Moods / 流行情绪
- Happy: upbeat, cheerful, celebration / 快乐：欢快、愉悦、庆祝
- Sad: ballad, heartbreak, nostalgia / 悲伤：抒情、心碎、怀旧
- Energetic: dance, anthem, party / 充满活力：舞曲、颂歌、派对

### Rock Moods / 摇滚情绪
- Aggressive: heavy, intense, fierce / 激进：沉重、强烈、凶猛
- Melancholic: alternative, grunge, introspective / 忧郁：另类、垃圾摇滚、内省
- Uplifting: classic, anthem, triumphant / 振奋：经典、颂歌、胜利

### Electronic Moods / 电子情绪
- Chill: ambient, lo-fi, relaxed / 轻松：氛围、低保真、放松
- Energetic: house, techno, driving / 充满活力：浩室、科技、驱动
- Dark: techno, industrial, ominous / 黑暗：科技、工业、不祥

### Jazz Moods / 爵士情绪
- Sophisticated: smooth, fusion, elegant / 精致：平滑、融合、优雅
- Melancholic: ballad, blues, reflective / 忧郁：抒情、布鲁斯、反思
- Upbeat: swing, bebop, energetic / 欢快：摇摆、比波普、充满活力

---

**Tip:** Use these patterns as starting points. Mix and match elements from different patterns to create unique genre combinations.
**提示：**将这些模式作为起点。混合和匹配来自不同模式的元素，创造独特的流派组合。
